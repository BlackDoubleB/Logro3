Tipos de listas en HTML

Para mas informacion puede dirigirse a la siguiente pagina Click aqui

En HTML, hay tres tipos principales de listas que se pueden crear utilizando las siguientes etiquetas:

Listas ordenadas (ol)

Las listas ordenadas se crean utilizando la etiqueta <ol> y contienen elementos de lista que se numeran automáticamente.

Ejemplo de una lista ordenada:

  1. Elemento 1.
  2. Elemento 2.
  3. Elemento 3.

Listas desordenadas (ul)

Las listas desordenadas se crean utilizando la etiqueta <ul> y contienen elementos de lista que se muestran con viñetas o puntos automáticamente.

Ejemplo de una lista desordenada:

  • Elemento A.
  • Elemento B.
  • Elemento C.

Listas de definición (dl)

Las listas de definición se crean utilizando la etiqueta <dl> y se componen de pares de términos (definiciones) que se muestran juntos.

Ejemplo de una lista de definición:

Término 1:
Definición 1.
Término 2:
Definición 2.
Término 3:
Definición 3.
